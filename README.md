# Versão de Negócios — Website Institucional

Site institucional da **Versão de Negócios**, empresa angolana especializada em soluções tecnológicas e de segurança em Luanda, Angola.

🌐 **Live:** [versaodenegocios.com](https://versaodenegocios.com)

---

## Sobre a Empresa

A Versão de Negócios oferece soluções integradas para empresas:

- **Controlo de Acessos** — Sistemas biométricos Suprema e ZKTeco
- **CFTV / Vigilância** — Câmeras IP Hikvision e Dahua 5MP, NVR 16 canais
- **Redes de Dados** — Cablagem FTP Cat.6, patch panels, certificação TIA/EIA-568
- **Audiovisual** — Projetores, som profissional, videoconferência
- **Marketing Digital** — Redes sociais, branding, campanhas, SEO
- **Azul Cowork** — Espaço de coworking com endereço fiscal em Luanda
- **Comércio Geral** — MacBooks, iPhones, equipamentos de TI e audiovisual
